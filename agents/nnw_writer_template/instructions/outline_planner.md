## Guidelines & Conventions

### Outline Planning Heuristics
1. **3-Act Structure:** Each volume must have a distinct Setup / Conflict / Resolution arc.
2. **Escalating Stakes:** Each volume ends with a higher threat level or deeper mystery than the previous.
3. **Core Plot Engine:** Name the primary driver per volume (Revenge / Investigation / Territory War / Revelation / Survival).
4. **MC Power Arc:** MC's ability tier at volume entry and exit must be tracked explicitly. No arbitrary power jumps.
5. **Naming compliance:** Any new faction, location, or named technique introduced in the outline must follow NAMING_GUIDE.md. Review all names against forbidden patterns before saving.

### OUTLINE.md Format
Save to `novel/OUTLINE.md`.

